Projects

  • Bus Tracking
    Website

    Technologies Used: HTML, CSS, JavaScript, Google Maps API

    I developed a comprehensive bus tracking website that revolutionizes the way students monitor and track their buses in real time. By leveraging the power of the Google Maps API, the website provides an intuitive and seamless interface for users to view bus routes and locations. The drivers' mobile locations are used to track buses, eliminating the need for additional GPS hardware. The website features an interactive map that highlights the Avadi location, providing live updates on bus positions. Additionally, the platform includes a real-time voice assistant that enhances user interaction by converting spoken commands into action. The project also implements a secure login system for both students and bus drivers, ensuring data privacy and accessibility only to authorized users.

  • URL Safety
    Checker

    Technologies Used: HTML, CSS, JavaScript, IP Geolocation APIs

    This project focuses on online safety by providing users with a platform to verify the safety of any URL before visiting it. The URL Safety Checker analyzes websites to determine whether they are safe to browse and detects the real physical location of the server hosting the website. Using IP Geolocation APIs, the site reveals crucial information about the server's geographic origin, giving users an added layer of protection against malicious websites. The project was designed with the modern web in mind, offering users a fast, easy-to-use interface to assess potential risks while browsing. This tool is particularly helpful in identifying phishing websites and preventing users from falling victim to online threats. By providing real-time results, the URL Safety Checker plays an essential role in promoting safe and secure browsing habits.

  • TrackMe - College Bus
    Tracking App

    Technologies Used: Android, GPS, Firebase

    TrackMe is a mobile application built to streamline college bus tracking using drivers' mobile locations, eliminating the need for costly GPS hardware. The app provides real-time tracking of buses, allowing students to view the exact location of their bus and receive accurate ETAs, making their commute more efficient and stress-free. Using the app, students can easily monitor bus routes, check schedules, and view updates on delays or changes in the route. By integrating Firebase, the app ensures real-time data updates and secure storage of user information. Drivers can log into the system to activate tracking, allowing students to see the bus’s live location on a map.

  • Music Synchronization
    App

    Technologies Used: Web Development (HTML, CSS, JavaScript), WebSockets, Node.js

    This exciting project is aimed at redefining the way people experience music with friends and loved ones. The Music Synchronization App allows users to listen to the same song in sync, regardless of where they are, creating a shared musical experience. The app synchronizes music playback between two users, allowing them to listen to the same track at the exact same time. It also includes a chat feature that activates only when both users are listening to the same song, encouraging real-time interaction and conversation around the music. Using WebSockets and Node.js, the app ensures seamless communication between devices, providing a smooth and engaging user experience.
